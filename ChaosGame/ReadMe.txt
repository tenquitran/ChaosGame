ChaosGame is a Win32 application for chaos game demonstration
(https://en.wikipedia.org/wiki/Chaos_game)

Uses libraries:
	- GLEW: http://glew.sourceforge.net/
	- GLM: http://glm.g-truc.net/

Created: February 13, 2018
